Server Routes

All server routes our app supports:

  • api/users | GET - Get All Users
  • api/users/user | GET - Get Active User
  • api/users/:id | GET - Get User
  • api/users/suggest/:id | GET - Get Friends Suggestions
  • api/users/activity/:id | PUT - Add User Activity
  • api/users/register | POST - Register
  • api/users/login | POST - Login
  • api/users/logout | POST - Logout
  • api/users/logout | POST - Logout
  • api/users/follow | POST - Follow User
  • api/users/unfollow | POST - Unfollow User
  • api/users/:id | DELETE - Delete User
  • api/posts/:userId | POST - Add Post
  • api/posts/getUserHomePosts/:userId | GET - Get User Homepage Posts
  • api/posts/:userId | GET - Get User Posts
  • api/posts | GET - Get All Posts
  • api/posts/like/:postId/:userId | PUT - Add Like To Post
  • api/posts/edit/:postId | PUT - Update Post Content
  • api/posts/:postId | DELETE - Delete Post
  • api/features | GET - Get Features
  • api/features | POST - Update Features
